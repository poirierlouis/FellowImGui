drag_indicator Drag and Drop

Create widgets in your document using drag and drop. Drag a widget from the toolbar and drop it in the tree. Insert widgets above and below widgets, or inside container widgets. You can move widgets around to reorder them.

menu_open More actions

Swipe a widget to the right to remove it. Right-click on a widget to show more actions. It includes duplicating a widget. You can use shortcuts to quickly execute an action on a selected widget.

navigation Navigation and focus

Click on a widget in the tree to focus it in the preview, and vice-versa. Use CTRL + click on a widget in the tree to scroll to in the preview, and vice-versa.

data_object Properties

Live update of properties of widgets. Select a widget in the tree or in the preview to edit its properties.
