drag_indicator Drag and Drop

Create widgets in your document using drag and drop. Drag a widget from the toolbar and drop it in the tree. Insert widgets above and below widgets, or inside container widgets. You can move widgets around to reorder them.

menu_open More actions

Swipe a widget to the right to remove it. Right-click on a widget to show more actions. It includes duplicating a widget. You can use shortcuts to quickly execute an action on a selected widget.

navigation Navigation and focus

Click on a widget in the tree to focus it in the preview, and vice-versa. Use CTRL + click on a widget in the tree to scroll to in the preview, and vice-versa.

data_object Properties

Live update of properties of widgets. Select a widget in the tree or in the preview to edit its properties.

format_color_fill Templates

Define styles of ImGui to replicate theme you are using in your project. Customize fonts, sizes and colors. You can embed custom fonts in documents. You can export templates as files to share with your team. Load a template to change styles of the current document.

widgets Widgets

Includes support for almost all ImGui widgets. It provides custom bloc widgets to quickly prototype a layout. For example, you can repeat widgets using a for-bloc widget. Bloc widgets are also generated in code.

@for (widget of widgets; track widget.type) { }

language Languages

Generate ready-to-use code based on the language binding you're using. You can generate code from a widget with a right-click. You can also use CTRL + G on a selected widget.

Widgets not supported by a language binding are highlighted. There is also partially supported widgets which fallback to a legacy API.

new_releases Compatibility

Fellow ImGui supports Dear ImGui v1.86 thanks to imgui-js, for now. Not all properties nor all widgets are implemented.
