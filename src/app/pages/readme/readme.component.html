drag_indicator Drag and Drop

Create widgets in your document by using drag and drop. Drag a widget from the toolbar and drop it into the tree. You can insert widgets above, below, or inside container widgets. Additionally, you can move widgets around to reorder them.

menu_open More actions

Swipe a widget to the right to remove it. Right-click on a widget to show more actions. It includes duplicating a widget. You can also use shortcuts to quickly perform actions on a selected widget.

navigation Navigation and focus

Click on a widget in the tree to focus it in the preview, and vice versa. Use CTRL + click on a widget in the tree to scroll to it in the preview, and vice versa.

data_object Properties

Provides live updates for widget properties. Select a widget in the tree or in the preview to edit its properties in real-time.

format_color_fill Templates

Define ImGui styles to replicate the theme used in your project. Customize fonts, sizes, colors, and embed custom fonts in documents. Export templates as files to share with your team, and load a template to update the styles of the current document.

widgets Widgets

Includes support for almost all ImGui widgets. It provides custom bloc widgets to quickly prototype a layout. For example, you can use a for-bloc widget to repeat widgets. These bloc widgets are also generated in the code.

@for (widget of widgets; track widget.type) { }

language Languages

Generate ready-to-use code based on the language binding you are using. You can generate code from a widget by right-clicking on it. Alternatively, you can use CTRL + G on a selected widget.

Widgets not supported by a language binding are highlighted. Partially supported widgets fallback to a legacy API.

information Compatibility

Fellow ImGui currently supports Dear ImGui v1.86 through imgui-js. However, not all properties and widgets are implemented at this time.
